## 📋 What is great-product-skills?

great-product-skills is a set of tools designed to guide you through the entire product development process. It helps you move from an initial idea all the way to a working prototype. The tools are built to work together, making the process smoother and easier to manage without needing any technical skills.

This package has four main steps, each handled by a different skill:

- **Product Discussion:** Talk through your ideas clearly.
- **Write Spec:** Create a detailed plan for your product.
- **Build Demo:** Develop the actual user interface.
- **UX Walkthrough:** Review and improve the user experience.

You don’t need to write code or understand software development deeply. Each skill acts like a helper to take care of the complex parts, so you can focus on making your product better.

---

## 🔍 How It Works

The key skill in great-product-skills is called **product-team**. It acts as an AI team lead, managing the full process and connecting the other skills. Here is what happens:

1. **Phase 1: Product Discussion** (skill: pm-debate)  
   Use this to discuss your product idea, ask questions, and make decisions on key features. It mimics a senior product manager who helps you think through the project.

2. **Phase 2: Write Spec** (skill: spec-generate)  
   This step creates a structured product requirement document (PRD). It guides you to write down what your product needs to do, step-by-step.

3. **Phase 3: Build Demo** (skills: frontend-design, web-artifacts-builder)  
   Here, the tools help turn your plan into a working frontend interface, ready to show how your product will look and behave.

4. **Phase 4: UX Walkthrough** (skill: ux-walkthrough)  
   Finally, review your product’s usability. Improve design and interaction until your demo feels right.
